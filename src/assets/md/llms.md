# Who we are

We are PeteZah Games, a community-driven gaming platform hosting over 400+ games
that are completely self-hosted for unrestricted access. Our mission is to
provide an unblocked gaming experience for users in restricted environments,
supported by a comprehensive proxy system.

## What we do

PeteZah Games operates as both a gaming hub and a proxy service:

- **Game Hosting**: We self-host over 400 games from reliable sources including
  Math 66 Games, 3kh0, and legacy Selenete.cc content
- **Proxy Services**: We provide unrestricted web access through our custom
  proxy infrastructure
- **Community Support**: We maintain active engagement with our 300k+ user base
  through our Discord community

## Our tech stack

- **Frontend**: Astro framework with React components, TypeScript, and Tailwind
  CSS
- **Backend**: Node.js with Express and Fastify for API endpoints
- **Authentication**: Supabase and SQL for account management and session
  handling
- **Proxy Infrastructure**: Titanium Network's Ultraviolet proxy and Scramjet
- **Storage & Database**: SQL for data persistence and user management
- **Deployment**: Multi-platform support including Vercel, Railway, Render, and
  custom servers

## Core features

- **400+ Games**: Comprehensive collection of unblocked games
- **Self-Hosted**: Complete control over game hosting and infrastructure
- **Proxy Support**: Multiple proxy protocols (Ultraviolet, Baremux, Epoxy,
  Scramjet)
- **User Accounts**: Persistent accounts with SQL backend
- **Social Integration**: Discord community and Twitter/X presence
- **Performance Monitoring**: Real-time FPS and battery monitoring for users
- **Mobile Ready**: Responsive design supporting desktop and mobile gaming
